Проєкт SkinSwap — Платформа для безпечного обміну ігровими предметами

1. Проблема та Пітч
Проблема: Гравці в багатокористувацькі ігри (CS, Roblox) часто стають жертвами шахрайства під час спроб обміну предметами поза офіційними майданчиками.
Пітч: SkinSwap — це вебсервіс, який гарантує безпеку угод завдяки системі Escrow (утримання предметів сервісом). Гравці отримують захищений інтерфейс для обміну віртуальними активами та прозору оцінку вартості предметів.

2. Ціль та MVP (Ядро цінності)
Ціль першої ітерації: Створити робочий прототип системи для безпечної передачі одного предмета між акаунтами.
MVP сценарій: Користувач авторизується через ігровий акаунт, обирає предмет та створює пропозицію, яка блокується системою до моменту фінального підтвердження обома сторонами .

3. Карта стейкхолдерів
Головними стейкхолдерами є геймери, які отримують цінність від безпеки своїх активів. Модератори платформи оцінюють успішність через відсутність конфліктів, а розробники працюють в межах технічних обмежень API ігрових платформ та часових ресурсів.

4. Сценарій користувача
Умови початку: Користувач увійшов у систему через ігровий профіль і має предмети в інвентарі .
Кроки:
Вибір предмета для обміну у власному інвентарі.
Генерація та надсилання посилання-запрошення іншому гравцеві.
Взаємне підтвердження умов угоди в інтерфейсі сервісу.
Очікуваний результат: Успішне завершення обміну або безпечне повернення предметів у разі скасування.

5. Критерії успіху
Технічна стабільність: Успішне виконання головного сценарію без критичних збоїв протягом тижня тестування.
Зручність: Здатність нового користувача завершити обмін самостійно без додаткових підказок.
Ефективність: Час виконання основної операції (створення обміну) не перевищує 2 хвилин.

6. План перевірки гіпотез
Ми перевіримо гіпотезу про те, що геймери готові до додаткових кроків підтвердження заради гарантованої безпеки. План включає аналіз відсотка завершених сценаріїв серед перших 50 тестових угод та збір стислих відгуків щодо відчуття безпеки.
